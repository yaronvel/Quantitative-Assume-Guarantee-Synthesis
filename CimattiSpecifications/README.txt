This folder contains unrealizable specifications from 

Alessandro Cimatti, Marco Roveri, Viktor Schuppan, Andrei Tchaltsev:
Diagnostic Information for Realizability. VMCAI 2008: 52-67

We have used these specifications to evaluate maximal realizability
shown in Table 2.

The specifications have been translated to the input format of out tool.
The reduction of GR(1)[F] specifications expressing maximal realizability
for these unrealizable GR(1) specifications is automated in our tool.
